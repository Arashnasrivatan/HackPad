# Macropad V1

This is a small macropad with 6 mechanical switches and a rotary encoder  
Its made to make common actions easier like copy, paste, cut, switching windows and ...

## Purpose

I built this for Hack Club  
I wanted to learn how to make the macropad from the beginning  
PCB design --> KiCad  
3D modeling --> Fusion 360  
Firmware --> Python KMK

My focus was on learning
